Binary Internals Viewer
=========================

An application to show binary file in structured way.

Download
 - Download Application: https://github.com/amosshi/freeinternals/releases

Supported Formats
 - BMP file
 - JPEG file
 - PNG file
 - ZIP file
 - DEX file (todo)
 - ELF file (todo)
 - PE file (TODO)
